## Project Description
This was my first Arduino project. The goal was to learn the basics of programming an Arduino by making an LED blink on and off repeatedly.

## Features
- LED blinks every second
- Uses digital output
- Demonstrates Arduino program structure

## Components Used
- Arduino Uno
- Breadboard
- LED
- 220Ω resistor
- Jumper wires
- USB cable

## How It Works
The Arduino sends a HIGH signal to digital pin 13 to turn the LED on. After one second it sends a LOW signal to turn the LED off. This repeats forever.

## Arduino Concepts Learned
- `setup()`
- `loop()`
- `pinMode()`
- `digitalWrite()`
- `delay()`
- Digital output pins

## Pin Setup

| Component | Arduino Pin |
|-----------|-------------|
| LED | Pin 13 |
